# SCHOOL-MANAGEMENT-SYSTEM

It helps in managing classes and timetables. It also manages students admission process.

## Running Tests

To run tests, run the following command

```bash
  clone repo
```
```bash
  install node
```
```bash
  npm init
```
```bash
  import sql file to mysql
```
